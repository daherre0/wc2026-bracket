# World Cup 2026 — Third-Place Standings (live)

**🔗 Live app: https://daherre0.github.io/wc2026-bracket/**

A single, self-contained static page that shows the FIFA World Cup 2026
third-place standings live, plus a live knockout-bracket tree.

- **Data:** ESPN public API (keyless, CORS-enabled) — no backend.
- **Third-place table:** toggle between ESPN official and a live-calculated table.
- **Knockout bracket:** tree view with local kick-off times + venues, and a
  "Projected (live)" mode that fills slots only from games already played or in
  progress (frozen at the current score); upcoming matches stay TBD until kickoff.

Open `index.html` locally, or use the live app linked above.
